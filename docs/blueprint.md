# **App Name**: PromptForge

## Core Features:

- Optimizer Selection: Allows users to select from a menu of pre-configured Optimizers, each tailored for specific content generation tasks.
- Chat Interface: Provides a chat-like interface where users can input their ideas or instructions.
- AI Content Generation: Generates content optimized according to the selected Optimizer's style, rules, and knowledge base.
- Admin Dashboard: Provides a central admin interface for configuring and managing Optimizers.
- Model Configuration: Enables administrators to configure the AI model (e.g., OpenAI, Anthropic), parameters (e.g., temperature, max tokens), and optional backup models or pipelines for each Optimizer.
- Knowledge Base Integration: Allows administrators to build modular knowledge bases for each Optimizer, composed of independent blocks like value propositions, key concepts, and content examples.
- Testing tool: Enables administrators to test Optimizers with example inputs, view the AI's response, inspect the complete prompt sent to the model, and iterate on adjustments.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to evoke a sense of innovation and efficiency.
- Background color: Light gray (#F0F2F5) for a clean and professional look.
- Accent color: A vivid orange (#FF9933) to highlight key actions and CTAs.
- Body and headline font: 'Inter', a sans-serif font with a modern, neutral, objective look, will be used for both headlines and body text.
- Use a set of minimalist icons to represent different Optimizer categories and functions within the platform.
- A clean, card-based layout with clear sections for each Optimizer setting.
- Subtle animations on input fields and button clicks to enhance the user experience.
